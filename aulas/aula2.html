🔩 A Anatomia de um Programa Java

1. Estrutura Geral de um Arquivo Java

Um arquivo .java pode conter várias partes. Em programas simples, usamos apenas a classe e o método main, mas é importante conhecer a estrutura completa:

  • Package: Define o pacote da classe. Ex.: package br.edu.unifametro;
  • Imports: Traz classes externas. Ex.: import java.util.Scanner;
  • Classe: Contém atributos e métodos. Ex.: public class Programa { }
  • Método main: Ponto de entrada. Ex.: public static void main(String[] args)

2. Estrutura Mínima de um Programa

Todo programa executável em Java precisa de, no mínimo, uma classe pública e um método main. Pense na classe como o "container" do código e no método main como a porta de entrada.

  • public class NomeDaClasse { ... }: O nome do arquivo deve ser idêntico ao da classe.
  • { ... }: As chaves delimitam blocos de código.
  • ;: Finaliza uma instrução, como um ponto final em uma frase.

3. Tipos de Dados e Sintaxe Básica

O Java é uma linguagem fortemente tipada, ou seja, cada variável precisa ter um tipo definido. Os mais comuns são:

  • int: números inteiros.
  • double: números com ponto flutuante.
  • boolean: valores lógicos (true ou false).
  • char: um único caractere.
  • String: sequência de caracteres (não é primitivo, mas usado como tal).

Dica: Use nomes de variáveis que descrevam o propósito. Ex.: idadeAluno é melhor que x.

4. O Método main

A assinatura public static void main(String[] args) é onde tudo começa. A JVM procura por esse método como ponto de entrada.

  • public: Acessível de qualquer lugar.
  • static: Pertence à classe, não a uma instância.
  • void: Não retorna valor.
  • String[] args: Recebe argumentos da linha de comando.

5. Entrada e Saída de Dados

Programas interativos precisam exibir informações e ler dados do usuário:

  • Saída: System.out.println() imprime no console.
  • Entrada: A classe Scanner lê dados do teclado. Exige o import import java.util.Scanner;.

💻 Exemplos Práticos

Exemplo 1: O Clássico "Hello, World!"

Nosso primeiro programa. Ele apenas imprime uma mensagem no console. 


/**
 * A classic "Hello, World!" program in Java.
 * This demonstrates the basic structure of a Java class with the main method.
 */
public class HelloWorld {
    public static void main(String[] args) {
        // Prints the text "Hello, LPOO World!" to the console.
        System.out.println("Hello, LPOO World!");
        
        // Using printf for formatted output (no line break by default).
        System.out.printf("Welcome to %s, Aula %d\n", "LPOO", 2);
    }
}

Exemplo 2: Lendo o Nome do Usuário

Este programa usa a classe Scanner para ler um nome digitado pelo usuário e depois exibe uma saudação personalizada. 


// Import the Scanner class from the java.util package to read user input.
import java.util.Scanner;

/**
 * A program that reads the user's name and prints a personalized greeting.
 */
public class Greeting {
    public static void main(String[] args) {
        // Create a Scanner object to read input from the console.
        Scanner inputReader = new Scanner(System.in);

        System.out.print("Please, enter your name: ");
        // Read the next line entered by the user.
        String userName = inputReader.nextLine();

        // Print a personalized greeting using printf.
        System.out.printf("Hello, %s! Welcome to LPOO.\n", userName);

        // Always close the Scanner to release resources.
        inputReader.close();
    }
}

Exemplo 3: Somando Dois Números

Aqui, lemos dois números inteiros, realizamos a soma e exibimos o resultado. 


import java.util.Scanner;

/**
 * A program that reads two integers, validates input, and prints their sum.
 */
public class SimpleCalculator {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        System.out.print("Enter the first number: ");
        int number1 = scanner.nextInt();

        System.out.print("Enter the second number: ");
        int number2 = scanner.nextInt();

        int sum = number1 + number2;

        // Using printf to format the output.
        System.out.printf("The sum of %d and %d is: %d\n", number1, number2, sum);

        // Basic validation example (just a demonstration)
        if(sum < 0) {
            System.out.println("Note: The sum is negative.");
        }

        scanner.close();
    }
}

Atividade Prática (TED): Cadastro Simples

Vamos criar um programa que simula um cadastro simples, coletando nome e idade do usuário e exibindo uma mensagem formatada. Esta atividade é o primeiro passo para a APS, onde construiremos sistemas que gerenciam dados.

  1. Crie uma nova classe chamada SimpleRegistration.java.
  2. Dentro do método main, crie um objeto Scanner para ler a entrada do teclado.
  3. Peça para o usuário digitar seu nome completo e armazene em uma variável String.
  4. Em seguida, peça para o usuário digitar sua idade e armazene em uma variável int.
  5. No final, imprima uma frase formatada no console, como: Cadastro realizado! Olá, [Nome], você tem [Idade] anos.
  6. Não se esqueça de fechar o Scanner no final do programa.

Sugestão de Metodologia Ativa

Tente resolver esta atividade em dupla (*pair programming*). Um colega digita ("piloto") enquanto o outro orienta e revisa ("co-piloto"). Troquem de papéis no meio da atividade. Isso ajuda a encontrar erros e a aprender mais rápido!

  • Utilizar a classe Scanner para ler dados do teclado.
